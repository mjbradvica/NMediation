# Security Policy

## Supported Versions

The current versions are supported; NMediation follows Microsoft's policy.

| Version | Dotnet | Supported          |
| ------- | ------ | ------------------ |
| 1.x.x   | 8/9/10 | :white_check_mark: |
| 0.9.x   | 8/9/10 | :white_check_mark: |

## Reporting a Vulnerability

NMediation will be normally updated on each dotnet major update.

The exception being if a sub-package has a vulnerability.

Please create an issue to make a report.
